# Fondamenti Java

## Argomenti

- Classi e oggetti
- Variabili e tipi primitivi
- Costruttori e scope
- Flusso di esecuzione (if, switch, cicli)
- Astrazione, interfacce e classi astratte
- Lambda expressions e generics
- Passaggio per valore e riferimento
- Operatori, espressioni booleane, short-circuit
